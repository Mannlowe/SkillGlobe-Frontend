# SkillGlobe - AI-Driven Talent Ecosystem

SkillGlobe is a modern web application built with Next.js that connects talent with opportunities in a seamless, AI-driven platform.

## Features

- **User Authentication**: Secure login and registration system
- **Onboarding Flow**: Personalized onboarding for both individuals and businesses
- **Profile Management**: Comprehensive profile creation and management
- **Mobile-Friendly**: Responsive design with mobile-specific optimizations
- **International Phone Support**: Country code selection with flags for phone number input

## Tech Stack

- **Framework**: Next.js 14
- **Styling**: Tailwind CSS
- **UI Components**: Custom components with Lucide icons
- **Form Handling**: React state management
- **Phone Input**: react-phone-input-2 for international phone numbers
- **Fonts**: Google Fonts (Rubik)

## Getting Started

### Prerequisites

- Node.js 18.x or higher
- npm or yarn

### Installation

1. Clone the repository
   ```bash
   git clone https://github.com/yourusername/skillglobe.git
   ```

2. Install dependencies
   ```bash
   cd skillglobe
   npm install
   # or
   yarn install
   ```

3. Run the development server
   ```bash
   npm run dev
   # or
   yarn dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser

## Project Structure

- `/app`: Next.js app router pages and layouts
- `/components`: Reusable React components
- `/public`: Static assets including images
- `/styles`: Global CSS and styling utilities
